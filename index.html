Sorting Algorithms Visualizer

About the Project

This project is basicaly a tool, made with python and pygame, for visualizing sorting algorithms in a educational way. Our objective is to demonstrate these algorithms so the user can understand how a computer can "move some pieces" to achive its goal, having a sorted data set at the end!

To accomplish our purpose we are also working on a wiki page with explanations for the algorithms and visualization GIFS, where the reader can gain a better understanding about the topic.
